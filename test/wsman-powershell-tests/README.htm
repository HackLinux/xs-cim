XenServer CIM WS-Management PowerShell scripted tests

This is a readme of the PowerShell scripts developed for the XenServer CIM project.

The aim of the PowerShell / WS-Management scripts was twofold:  to validate the XenServer CIM WS-Management interface and to provide feedback to the feature set as a developer attempting to perform common actions against a virtualization environment using the API.

The following outlines each PowerShell script and its intent:

Script

Description

AutomatedTestWsmanXenCim.ps1

This script is a mimic of the AutomatedTestCore PowerShell script that is installed with the XenServer PowerShell cmdlets.

This served as a basis for actions considered important by the XenServer team.

CreateInstallVmWsmanXenCim.ps1

This script was originally designed to simulate the administrative actions of Creating a new VM, attaching an ISO, booting the VM, connecting to the console then detaching the ISO, and destroying everything.

The console connection portion of this script has not been completed because the ActiveX control is ‘not scriptable’.

CreateXxVmsFromTemplate.ps1

This script is designed to take a single template and duplicate that into the number of VMs that is set in the loop.

This can be used to provide stress as well as build out environments for enumeration testing.

DataRefreshers.ps1

This script is designed to simulate the background polling that is constant from the SCVMM data refreshers.

This can be run on its own in in conjunction with another script to simulate actions alongside of data refresher activity. 

The data refreshers as also triggered on an on demand basis by selecting elements in the SCVMM console.

ExportAndImportVM.ps1

This script was written to simulate the SCVMM action of saving a VM to the SCVMM Library and then deploying a VM from the SCVMM Library share using BITS for disk copy.

This test has had a loop added so it can be used to apply continuous stress.

ISOCopyTestWsmanXenCim.ps1

This script was written to simulate the SCVMM action of deploying and using an ISO from the SCVMM Library.

BITS is used for disk copy as well as a CIFS ISO Storage Repository.

This test has had a loop added so it can be used to apply continuous stress.

WSMAN.EnumAll.ps1

This script very simply performs an enumeration against all CIM Methods and classes.

Association classes are not supported through WSMAN.

This has been useful in determining if methods have been implemented.

XenWsMan.Functions.ps1

This is a very large script that contains helper functions for the other scripts mentioned above as well as examples of actions.

Each action within this script needs to be manually tested.  This script serves as a set of examples and methods for other scripts as well as the original proving ground for performing many common functions using the WSMAN interface.
